# Quick Start Guide

This guide will walk you through creating your first advertising display with PromoMaster Pro. You'll learn how to set up a store, add products, and launch your first display sequence in just a few minutes.

<Note>
PromoMaster Pro comes with a default store ("Sucursal Central") already configured, so you can start adding products immediately.
</Note>

## Getting Your First Display Live

<Steps>
  <Step title="Open the Application">
    Launch PromoMaster Pro in your browser. You'll see the **Panel de Control** (Dashboard) with two main navigation items:
    - **Panel de Control**: Create and launch display sequences
    - **Gestión Total**: Manage stores, products, and announcements

    The dashboard is your command center for creating advertising displays. Start by familiarizing yourself with the three tabs:
    - **Productos**: Your product catalog
    - **Avisos Libres**: Custom text announcements
    - **Mi Secuencia**: Your display playlist
  </Step>

  <Step title="Create Your First Store (Optional)">
    If you want to add additional store locations, navigate to **Gestión Total** from the sidebar.

    Click the store management section and add a new store with:
    ```typescript
    interface Store {
      name: string;        // e.g., "Downtown Branch"
      address: string;     // e.g., "123 Main Street"
      logoColor: string;   // e.g., "bg-blue-600"
    }
    ```

    Each store can have its own branding color that appears on displays. The app will automatically generate a unique ID for your store.

    <Tip>
    The `logoColor` field uses Tailwind CSS background color classes. Popular options include: `bg-blue-600`, `bg-green-600`, `bg-purple-600`, `bg-red-600`.
    </Tip>
  </Step>

  <Step title="Add Your First Product">
    Navigate to **Gestión Total** and go to the products section. Click to add a new product.

    Here's the product data structure you'll be working with:
    ```typescript
    interface Product {
      name: string;              // Product name
      description: string;       // Detailed description
      price: number;            // Price in your currency
      unit: UnitType;           // 'unidad' | 'kg' | 'g' | 'litro' | 'pack'
      category: string;         // e.g., "Beverages", "Dairy", "Produce"
      imageUrl?: string;        // Optional product image URL
      isPromotion: boolean;     // Flag for promotional items
      promotionType?: PromotionType;  // 'dia' | 'semana' | 'general'
      slogan?: string;          // Optional marketing slogan
      primaryColor?: string;    // Hex code for display theme
    }
    ```

    **Example Product:**
    - **Name**: Fresh Organic Apples
    - **Description**: Crisp, sweet apples from local farms
    - **Price**: 3.99
    - **Unit**: kg
    - **Category**: Produce
    - **Is Promotion**: Yes
    - **Promotion Type**: semana (weekly special)

    <Tip>
    Leave the `imageUrl` empty to use auto-generated placeholder images, or provide a URL to your product image.
    </Tip>
  </Step>

  <Step title="Create a Custom Announcement (Optional)">
    Announcements are perfect for displaying text-based messages alongside your products.

    ```typescript
    interface Announcement {
      title: string;              // Main headline
      message: string;            // Full message text
      fontSize: number;           // Text size in pixels
      fontFamily: FontType;       // 'font-bebas' | 'font-sans' | 'font-serif' | 'font-mono'
      backgroundColor: string;    // Hex color code
      textColor: string;          // Hex color code
      textAlign: 'left' | 'center' | 'right';
      hasAnimation: boolean;      // Enable text animations
    }
    ```

    **Example Announcement:**
    - **Title**: Weekend Sale!
    - **Message**: Save up to 30% on all fresh produce this weekend only
    - **Font Size**: 48
    - **Font Family**: font-bebas
    - **Background Color**: #4F46E5 (indigo)
    - **Text Color**: #FFFFFF (white)
    - **Text Align**: center
    - **Has Animation**: true
  </Step>

  <Step title="Build Your Display Sequence">
    Return to the **Panel de Control** dashboard. Here's where the magic happens:

    1. **Select Your Store**: Use the store dropdown in the top-right to choose which location will display the content

    2. **Set Display Duration**: Use the timer control to set how long each item appears (default: 5 seconds)

    3. **Add Items to Your Sequence**:
       - Browse the **Productos** tab and click the **+** button on products you want to feature
       - Switch to **Avisos Libres** tab and add announcements
       - Use the "Añadir Todo a Secuencia" button to add everything at once

    4. **Arrange Your Sequence**: Switch to the **Mi Secuencia** tab to:
       - Reorder items with up/down arrows
       - Remove items you don't want
       - Preview the complete sequence

    The display will show items in the order they appear in your sequence, transitioning automatically based on your duration setting.
  </Step>

  <Step title="Launch Your Display">
    Once your sequence is ready:

    1. Click the **"Iniciar Transmisión"** button in the Mi Secuencia tab to start the full sequence
    2. Alternatively, click the **Play** button on any individual product or announcement to launch just that item

    Your display will enter fullscreen mode and begin cycling through your items. Each item will show for your configured duration before transitioning to the next.

    <Warning>
    Once in display mode, the app takes over the full screen. Press ESC or click the exit button to return to the dashboard.
    </Warning>
  </Step>
</Steps>

## Understanding Display Items

The sequence system uses a unified item structure:

```typescript
interface DisplayItem {
  type: 'product' | 'announcement';
  id: string;  // References the product or announcement ID
}
```

This allows you to freely mix products and announcements in any order, creating dynamic advertising experiences.
