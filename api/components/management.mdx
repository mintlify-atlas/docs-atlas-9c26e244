## Overview

The Management component provides a comprehensive interface for creating, editing, and deleting stores, products, and announcements in PromoMaster Pro. It includes AI-powered features for content generation.

## Features

### Product Management

- **Create/Edit Products**: Full product information form
- **Image Management**: Three methods for adding product images:
  - URL input
  - File upload
  - AI generation using Gemini
- **AI Slogan Generation**: Generate marketing slogans automatically
- **Promotion Configuration**: Mark products as promotions with type selection
- **Pricing and Units**: Configure price and unit of measurement

### Announcement Management

- **Visual Editor**: Create custom text announcements
- **Real-time Preview**: See changes instantly as you edit
- **Typography Controls**: Font family, size, and alignment
- **Color Customization**: Background and text color pickers
- **Animation Toggle**: Enable/disable animation effects

### Store Management

- **Basic Information**: Name and address
- **Brand Colors**: Logo color configuration
- **Edit/Delete**: Modify or remove existing stores

## Usage Example

```typescript
import Management from './components/Management';
import { Store, Product, Announcement } from './types';
import { v4 as uuidv4 } from 'uuid';

function App() {
  const [stores, setStores] = useState<Store[]>([]);
  const [products, setProducts] = useState<Product[]>([]);
  const [announcements, setAnnouncements] = useState<Announcement[]>([]);

  return (
    <Management
      stores={stores}
      products={products}
      announcements={announcements}
      onAddStore={(store) => 
        setStores([...stores, { ...store, id: uuidv4() }])
      }
      onUpdateStore={(store) =>
        setStores(stores.map(s => s.id === store.id ? store : s))
      }
      onDeleteStore={(id) =>
        setStores(stores.filter(s => s.id !== id))
      }
      onAddProduct={(product) =>
        setProducts([...products, { ...product, id: uuidv4() }])
      }
      onUpdateProduct={(product) =>
        setProducts(products.map(p => p.id === product.id ? product : p))
      }
      onDeleteProduct={(id) =>
        setProducts(products.filter(p => p.id !== id))
      }
      onAddAnnouncement={(ann) =>
        setAnnouncements([...announcements, { ...ann, id: uuidv4() }])
      }
      onUpdateAnnouncement={(ann) =>
        setAnnouncements(announcements.map(a => a.id === ann.id ? ann : a))
      }
      onDeleteAnnouncement={(id) =>
        setAnnouncements(announcements.filter(a => a.id !== id))
      }
    />
  );
}
```

## AI Features

### Generate Slogan

```typescript
const handleGenerateSlogan = async () => {
  if (!productForm.name) {
    alert("Indica el nombre del producto.");
    return;
  }
  setIsGeneratingSlogan(true);
  const slogan = await generateSlogan(
    productForm.name,
    productForm.description
  );
  setProductForm(prev => ({ ...prev, slogan }));
  setIsGeneratingSlogan(false);
};
```

### Generate Image

```typescript
const handleGenerateImage = async () => {
  if (!productForm.name) {
    alert("Introduce un nombre de producto para generar una imagen.");
    return;
  }
  setIsGeneratingImage(true);
  const imageUrl = await generateProductImage(productForm.name);
  if (imageUrl) {
    setProductForm(prev => ({ ...prev, imageUrl }));
  }
  setIsGeneratingImage(false);
};
```

### File Upload

```typescript
const handleFileUpload = (e: React.ChangeEvent<HTMLInputElement>) => {
  const file = e.target.files?.[0];
  if (file) {
    const reader = new FileReader();
    reader.onloadend = () => {
      setProductForm(prev => ({
        ...prev,
        imageUrl: reader.result as string
      }));
    };
    reader.readAsDataURL(file);
  }
};
```

## Tab Interface

The Management component provides a tabbed interface:

1. **Products**: Create and manage product catalog
2. **Announcements**: Create and manage custom announcements with live preview
3. **Stores**: Create and manage store locations

## Validation

<Warning>
**Required Fields**
- Products: name, price, category are required
- Announcements: title and message are required
- Stores: name and address are required
</Warning>

## Best Practices

<Tip>
**Edit Mode**
- Clicking edit on an item populates the form and scrolls to top
- Cancel button appears when editing to reset the form
- Form button text changes to indicate edit mode
</Tip>

<Tip>
**Image Management**
- AI image generation takes 15-30 seconds - show loading states
- File uploads are converted to base64 data URIs for easy storage
- URL mode allows linking to external images
</Tip>
