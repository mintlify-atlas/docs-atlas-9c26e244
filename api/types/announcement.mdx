## Overview

The `Announcement` interface represents a custom text-based message that can be displayed on screens. Announcements support full styling control including fonts, colors, alignment, and animations.

## Type Definition

```typescript
export type FontType = 'font-bebas' | 'font-sans' | 'font-serif' | 'font-mono';

export interface Announcement {
  id: string;
  title: string;
  message: string;
  fontSize: number; // in pixels
  fontFamily: FontType;
  backgroundColor: string;
  textColor: string;
  textAlign: 'left' | 'center' | 'right';
  hasAnimation: boolean;
}
```

## Properties

<ResponseField name="id" type="string" required>
  Unique identifier for the announcement
</ResponseField>

<ResponseField name="title" type="string" required>
  Internal title for managing announcements (not displayed to end users)
</ResponseField>

<ResponseField name="message" type="string" required>
  The actual message text that will be displayed on screen
</ResponseField>

<ResponseField name="fontSize" type="number" required>
  Font size in pixels. Typically ranges from 20 to 200
</ResponseField>

<ResponseField name="fontFamily" type="FontType" required>
  Font style for the announcement. Can be one of:
  - `'font-bebas'` - Bold, impactful display font
  - `'font-sans'` - Clean, modern sans-serif
  - `'font-serif'` - Elegant serif font
  - `'font-mono'` - Technical monospace font
</ResponseField>

<ResponseField name="backgroundColor" type="string" required>
  Background color as a hex code (e.g., `#1e293b`)
</ResponseField>

<ResponseField name="textColor" type="string" required>
  Text color as a hex code (e.g., `#ffffff`)
</ResponseField>

<ResponseField name="textAlign" type="'left' | 'center' | 'right'" required>
  Text alignment within the display area
</ResponseField>

<ResponseField name="hasAnimation" type="boolean" required>
  Whether to apply animation effects to the announcement
</ResponseField>

## Usage Example

```typescript
const announcement: Announcement = {
  id: 'ann-001',
  title: 'Summer Hours',
  message: 'Open 7 days a week from 8 AM to 10 PM',
  fontSize: 64,
  fontFamily: 'font-bebas',
  backgroundColor: '#1e293b',
  textColor: '#ffffff',
  textAlign: 'center',
  hasAnimation: true
};
```

## Font Types

| Font Type | Description | Best For |
|-----------|-------------|----------|
| `font-bebas` | Bebas Neue - Bold, impactful | Promotions, headlines |
| `font-sans` | Inter - Clean, modern | General announcements |
| `font-serif` | Elegant serif | Formal notices |
| `font-mono` | Technical monospace | Technical information |

## Styling Guidelines

<Tip>
For optimal readability on digital displays:
- Use high contrast between text and background colors
- Font sizes between 48-80px work well for most announcements
- Center alignment is recommended for short messages
- Enable animations sparingly for important announcements
</Tip>
