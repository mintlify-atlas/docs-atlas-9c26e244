## Overview

The `Store` interface represents a retail location in the PromoMaster Pro system. Each store contains basic information like name, address, and branding color.

## Type Definition

```typescript
export interface Store {
  id: string;
  name: string;
  address: string;
  logoColor: string;
}
```

## Properties

<ResponseField name="id" type="string" required>
  Unique identifier for the store
</ResponseField>

<ResponseField name="name" type="string" required>
  Display name of the store location
</ResponseField>

<ResponseField name="address" type="string" required>
  Physical address of the store
</ResponseField>

<ResponseField name="logoColor" type="string" required>
  CSS class or color value for store branding. Typically uses Tailwind CSS classes like `bg-indigo-600`
</ResponseField>

## Usage Example

```typescript
const store: Store = {
  id: 'store-001',
  name: 'Downtown Market',
  address: '123 Main Street, City Center',
  logoColor: 'bg-blue-600'
};
```

## Related Types
