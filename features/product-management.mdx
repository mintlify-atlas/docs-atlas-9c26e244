## Overview

The Product Management system allows you to build and maintain a complete catalog of retail products with rich media, pricing, promotional configurations, and AI-generated content.

## Product Data Structure

Every product in PromoMaster Pro contains the following information:

```typescript
export interface Product {
  id: string;
  name: string;
  description: string;
  price: number;
  unit: UnitType;              // 'unidad' | 'kg' | 'g' | 'litro' | 'pack'
  imageUrl?: string;
  category: string;
  isPromotion: boolean;
  promotionType?: PromotionType; // 'dia' | 'semana' | 'general'
  slogan?: string;
  primaryColor?: string;         // Hex code for the ad theme
}
```

## Image Management

### Three Image Source Options

#### URL Method
Paste a direct link to an image hosted online:

```tsx
<input 
  type="text" 
  placeholder="https://ejemplo.com/foto.jpg" 
  value={productForm.imageUrl}
  onChange={e => setProductForm({...productForm, imageUrl: e.target.value})}
/>
```

#### Upload Method
Upload images directly from your device. Images are converted to base64 data URLs:

```tsx
const handleFileUpload = (e: React.ChangeEvent<HTMLInputElement>) => {
  const file = e.target.files?.[0];
  if (file) {
    const reader = new FileReader();
    reader.onloadend = () => {
      setProductForm(prev => ({ ...prev, imageUrl: reader.result as string }));
    };
    reader.readAsDataURL(file);
  }
};
```

#### AI Generation
Generate product images using AI based on the product name:

```tsx
const handleGenerateImage = async () => {
  if (!productForm.name) {
    alert("Introduce un nombre de producto para generar una imagen.");
    return;
  }
  setIsGeneratingImage(true);
  const imageUrl = await generateProductImage(productForm.name);
  if (imageUrl) setProductForm(prev => ({ ...prev, imageUrl }));
  setIsGeneratingImage(false);
};
```

<Warning>
AI image generation requires a product name to be entered first.
</Warning>

## AI-Powered Slogans

Generate compelling advertising slogans automatically using AI:

```tsx
<button 
  type="button" 
  onClick={async () => {
    if(!productForm.name) return alert("Indica el nombre del producto.");
    setIsGeneratingSlogan(true);
    const s = await generateSlogan(productForm.name, productForm.description);
    setProductForm(prev => ({ ...prev, slogan: s }));
    setIsGeneratingSlogan(false);
  }}
>
  {isGeneratingSlogan ? <Loader2 className="animate-spin" /> : <Sparkles />}
</button>
```

<Tip>
AI-generated slogans are displayed prominently in the full-screen display mode beneath the product name.
</Tip>

## Editing & Deleting Products

### Edit Products
Click the edit icon on any product card to load it into the form. The form switches to "Update" mode:

```tsx
const handleProductSubmit = async (e: React.FormEvent) => {
  e.preventDefault();
  if (editingProductId) {
    onUpdateProduct({ ...productForm, id: editingProductId });
    setEditingProductId(null);
  } else {
    onAddProduct(productForm);
  }
  setProductForm(INITIAL_PRODUCT_STATE);
};
```

### Delete Products
Click the trash icon to remove a product from the catalog permanently.

<Warning>
Deleting a product is permanent and cannot be undone. Ensure products are not actively being displayed before deletion.
</Warning>

## Category & Color Management

### Product Categories
Organize products by category for easier searching and filtering in the Dashboard:

```tsx
const filteredProducts = products.filter(p => 
  p.name.toLowerCase().includes(searchTerm.toLowerCase()) || 
  p.category.toLowerCase().includes(searchTerm.toLowerCase())
);
```

### Theme Colors
Set a primary color for each product to customize the display theme:

```typescript
primaryColor: '#4f46e5'  // Indigo by default
```

This color affects:
- Background gradients in display mode
- Promotion badge colors
- Price accent colors
- Ticker text highlights
