## Overview

The Playlist Builder in the Dashboard allows you to create custom sequences of products and announcements that play in order during display mode. Mix content types, reorder items, and launch targeted advertising campaigns.

## Playlist Data Structure

Playlists are arrays of display items that reference products or announcements:

```typescript
export interface DisplayItem {
  type: 'product' | 'announcement';
  id: string;  // References Product.id or Announcement.id
}
```

```typescript
const [playlist, setPlaylist] = useState<DisplayItem[]>([]);
```

<Note>
Display items store only the type and ID. Full data is resolved when rendering or launching displays.
</Note>

## Adding Items to Playlist

### Single Item Addition

Click the + button on any product or announcement card:

```tsx
const addToPlaylist = (item: DisplayItem) => {
  setPlaylist(prev => [...prev, item]);
};

<button 
  onClick={() => addToPlaylist({ type: 'product', id: product.id })}
  title="Añadir a secuencia"
>
  <Plus size={20} />
</button>
```

### Bulk Addition

Add all products and announcements at once:

```tsx
const addAllToPlaylist = () => {
  const allProducts: DisplayItem[] = products.map(p => ({ 
    type: 'product', 
    id: p.id 
  }));
  const allAnns: DisplayItem[] = announcements.map(a => ({ 
    type: 'announcement', 
    id: a.id 
  }));
  setPlaylist([...allProducts, ...allAnns]);
  setActiveTab('playlist');
};

<button onClick={addAllToPlaylist}>
  <Plus size={18} /> Añadir Todo a Secuencia
</button>
```

<Tip>
Bulk addition automatically switches to the playlist tab so you can review and reorder items.
</Tip>

## Playlist Management

### Viewing the Playlist

The playlist tab shows all queued items in order:

```tsx
<div className="space-y-3">
  {playlist.map((item, index) => {
    const data = item.type === 'product' 
      ? products.find(p => p.id === item.id) 
      : announcements.find(a => a.id === item.id);
    if (!data) return null;

    return (
      <div key={`${item.id}-${index}`} className="flex items-center gap-4 p-4 bg-slate-50 rounded-2xl">
        <div className="text-xs font-black text-slate-300 w-6">{index + 1}</div>
        <div className="w-12 h-12 rounded-xl">
          {item.type === 'product' ? (
            <img src={(data as Product).imageUrl} />
          ) : (
            <Megaphone size={18} />
          )}
        </div>
        <div className="flex-1">
          <div className="font-bold">
            {item.type === 'product' ? (data as Product).name : (data as Announcement).title}
          </div>
          <div className="text-[10px] uppercase">
            {item.type === 'product' ? 'Producto' : 'Aviso Libre'}
          </div>
        </div>
      </div>
    );
  })}
</div>
```

### Reordering Items

Move items up or down in the sequence:

```tsx
const movePlaylistItem = (index: number, direction: 'up' | 'down') => {
  const newPlaylist = [...playlist];
  const targetIndex = direction === 'up' ? index - 1 : index + 1;
  if (targetIndex < 0 || targetIndex >= newPlaylist.length) return;
  [newPlaylist[index], newPlaylist[targetIndex]] = 
    [newPlaylist[targetIndex], newPlaylist[index]];
  setPlaylist(newPlaylist);
};

<button onClick={() => movePlaylistItem(index, 'up')}>
  <ArrowUp size={16} />
</button>
<button onClick={() => movePlaylistItem(index, 'down')}>
  <ArrowDown size={16} />
</button>
```

<Note>
Reordering is disabled for first item (can't move up) and last item (can't move down).
</Note>

### Removing Items

Delete individual items from the playlist:

```tsx
const removeFromPlaylist = (index: number) => {
  setPlaylist(prev => prev.filter((_, i) => i !== index));
};

<button onClick={() => removeFromPlaylist(index)}>
  <X size={16} />
</button>
```

### Clearing Playlist

Remove all items at once:

```tsx
<button onClick={() => setPlaylist([])}>
  Vaciar
</button>
```

## Launching Displays

### Launch Full Playlist

Start display mode with entire playlist sequence:

```tsx
const handleLaunchAllMixed = () => {
  if (playlist.length > 0 && selectedStore) {
    onStartDisplay(playlist, selectedStore, duration);
  }
};

<button 
  disabled={playlist.length === 0}
  onClick={handleLaunchAllMixed}
>
  <MonitorPlay size={20} /> Iniciar Transmisión
</button>
```

<Warning>
Ensure a store is selected and at least one item is in the playlist before launching.
</Warning>

### Launch Single Item

Quickly display one product or announcement:

```tsx
const handleLaunchItemNow = (item: DisplayItem) => {
  onStartDisplay([item], selectedStore, duration);
};

<button onClick={() => handleLaunchItemNow({ type: 'product', id: product.id })}>
  <Play size={20} fill="currentColor" />
</button>
```

<Tip>
Launching a single item bypasses the playlist and starts display mode immediately.
</Tip>

## Search & Filter

Find products and announcements quickly:

```tsx
const [searchTerm, setSearchTerm] = useState('');

const filteredProducts = products.filter(p => 
  p.name.toLowerCase().includes(searchTerm.toLowerCase()) || 
  p.category.toLowerCase().includes(searchTerm.toLowerCase())
);

const filteredAnnouncements = announcements.filter(a =>
  a.title.toLowerCase().includes(searchTerm.toLowerCase()) ||
  a.message.toLowerCase().includes(searchTerm.toLowerCase())
);

<input 
  type="text" 
  placeholder="Buscar en productos..."
  value={searchTerm}
  onChange={(e) => setSearchTerm(e.target.value)}
/>
```

Search works across:
- Product names and categories
- Announcement titles and messages

## Duration Configuration

Set how long each item displays:

```tsx
const [duration, setDuration] = useState(defaultDuration);

<div className="flex items-center gap-2">
  <Clock size={16} />
  <span>Segundos</span>
  <input 
    type="number" 
    min="1" 
    value={duration}
    onChange={(e) => setDuration(Math.max(1, parseInt(e.target.value) || 1))}
  />
</div>
```

Duration applies to all items in the playlist equally.

<Note>
Duration must be at least 1 second. The system enforces this minimum automatically.
</Note>

## Store Selection Context

Playlist launching requires store context:

```tsx
const [selectedStore, setSelectedStore] = useState<string>(stores[0]?.id || '');

<select 
  value={selectedStore}
  onChange={(e) => setSelectedStore(e.target.value)}
>
  {stores.map(s => (
    <option key={s.id} value={s.id}>{s.name}</option>
  ))}
</select>
```

The selected store determines:
- Store name shown in displays
- Address information in product displays
- Ticker messages

## Best Practices

<Accordion title="Strategic Ordering">
Place high-priority items early in the sequence when customer attention is highest. End with strong call-to-action announcements.
</Accordion>

<Accordion title="Playlist Length">
Keep playlists to 5-15 items for most scenarios. Longer sequences may lose viewer attention before cycling.
</Accordion>

<Accordion title="Content Balance">
Mix products and announcements for variety. Too many products can feel overwhelming; too many announcements can lack impact.
</Accordion>

<Accordion title="Regular Updates">
Refresh playlists frequently to reflect current inventory, promotions, and seasonal changes.
</Accordion>
