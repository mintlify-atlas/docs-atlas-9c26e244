## Overview

Display Mode transforms your selected products and announcements into professional, full-screen advertising displays with smooth animations, theme-aware styling, and attention-grabbing visual effects. Perfect for in-store digital signage and promotional screens.

## Display Architecture

The display system cycles through items with configurable timing:

```typescript
interface AdDisplayProps {
  items: (Product | Announcement)[]; // Mixed content array
  store: Store;                      // Current store location
  duration: number;                  // Seconds per item
  onExit: () => void;                // Exit callback
}
```

## Auto-Rotation System

Displays automatically cycle through your playlist:

```tsx
const nextSlide = useCallback(() => {
  setShowContent(false);
  setTimeout(() => {
    setCurrentIndex((prev) => (prev + 1) % items.length);
    setProgress(0);
    setShowContent(true);
  }, 500);
}, [items.length]);

useEffect(() => {
  if (items.length <= 1) return;

  const interval = 100;
  const totalSteps = (duration * 1000) / interval;
  
  const timer = setInterval(() => {
    setProgress((prev) => {
      if (prev >= 100) {
        nextSlide();
        return 0;
      }
      return prev + (100 / totalSteps);
    });
  }, interval);

  return () => clearInterval(timer);
}, [duration, items.length, nextSlide]);
```

<Note>
The rotation system includes a 500ms fade transition between items for smooth visual flow.
</Note>

## Ticker System

Continuous scrolling ticker at the bottom displays dynamic messages:

```tsx
const tickerMessages = useMemo(() => {
  if (isProduct && product) {
    const messages = [
      product.name.toUpperCase(),
      "CALIDAD GARANTIZADA",
      "DISPONIBLE AHORA",
      store.name.toUpperCase()
    ];
    if (product.isPromotion) {
      messages.push("¡OFERTA LIMITADA!");
      messages.push(`MEJOR PRECIO: $${product.price.toLocaleString()}`);
    }
    return messages;
  } else if (announcement) {
    return [
      announcement.title.toUpperCase(),
      "AVISO IMPORTANTE",
      store.name.toUpperCase(),
      "¡GRACIAS POR SU VISITA!"
    ];
  }
  return [];
}, [currentItem, store.name]);
```

```tsx
<div className="absolute bottom-4 left-0 w-full bg-black/40 backdrop-blur-md py-4 overflow-hidden">
  <div className="flex animate-marquee space-x-20 px-10">

      <React.Fragment key={i}>
        {tickerMessages.map((msg, idx) => (
          <span 
            key={`${i}-${idx}`}
```

## Slide Indicators

Interactive dots showing position in playlist:

```tsx
{items.length > 1 && (
  <div className="absolute bottom-20 left-1/2 -translate-x-1/2 flex gap-3">
    {items.map((_, i) => (
      <div 
        key={i} 
        className={`h-2 transition-all duration-700 rounded-full cursor-pointer ${
          i === currentIndex ? 'w-12 bg-white' : 'w-2 bg-white/10'
        }`}
        style={{ backgroundColor: i === currentIndex ? themeColor : undefined }}
        onClick={() => {
          setCurrentIndex(i);
          setProgress(0);
          setShowContent(false);
          setTimeout(() => setShowContent(true), 100);
        }}
      ></div>
    ))}
  </div>
)}
```

<Note>
Users can click indicators to jump directly to specific items in the playlist.
</Note>

## Theme Color System

Each display adapts to product or announcement colors:

```tsx
const themeColor = product 
  ? (product.primaryColor || '#4f46e5') 
  : (announcement?.backgroundColor || '#1e293b');
```

The theme color affects:
- Background gradient blobs
- Progress bar color
- Currency symbol color
- Ticker text highlights
- Slide indicator active state
- Promotion badge backgrounds
- Store badge backgrounds

## Duration Controls

Display duration is configurable before launching:

```tsx
<div className="flex items-center gap-2">
  <Clock size={16} />
  <span className="text-xs font-bold">Segundos</span>
  <input 
    type="number" 
    min="1" 
    value={duration}
    onChange={(e) => setDuration(Math.max(1, parseInt(e.target.value) || 1))}
  />
</div>
```

Recommended durations:
- **3-5 seconds**: Fast-paced, high-traffic areas
- **7-10 seconds**: Standard retail display
- **15+ seconds**: Detailed products or longer announcements
