# Welcome to PromoMaster Pro

PromoMaster Pro is a powerful SaaS application designed for retail businesses to create, manage, and display professional advertising content across multiple store locations. Built with React and TypeScript, it provides an intuitive interface for managing products, announcements, and coordinated display sequences.

## What is PromoMaster Pro?

PromoMaster Pro transforms how retailers manage their in-store advertising displays. Instead of manually updating promotional materials or relying on static signage, you can dynamically create and broadcast product promotions and custom announcements to digital displays in your stores.

## Who is it for?

- **Retail Store Managers**: Quickly update promotional content across multiple locations
- **Marketing Teams**: Create engaging product displays and announcements with custom styling
- **Multi-store Chains**: Coordinate advertising campaigns across all store locations from a central dashboard
- **Small Businesses**: Professional advertising displays without expensive design software

## Key Capabilities

**Product Management**
- Create and manage product catalogs with prices, categories, and images
- Support for various unit types (kg, g, litro, pack, unidad)
- Promotion flags and custom styling options
- AI-powered content generation for descriptions and slogans

**Dynamic Announcements**
- Custom text-based announcements with full styling control
- Configurable fonts, colors, text alignment, and animations
- Perfect for special offers, store hours, or important messages

**Display Sequences**
- Create custom playlists mixing products and announcements
- Configurable display duration per item
- Drag-and-drop reordering for complete control
- Launch individual items or full sequences

**Multi-Store Support**
- Manage multiple store locations from one dashboard
- Store-specific branding with custom colors
- Coordinate displays across all locations

## Built With Modern Technology

PromoMaster Pro is built with cutting-edge web technologies:
- **React 19** for a responsive, modern interface
- **TypeScript** for type-safe development
- **Vite** for lightning-fast development and builds
- **Google GenAI** for intelligent content generation
- **Local Storage** for persistent data management

<Note>
PromoMaster Pro stores all data locally in your browser, giving you complete control over your content without requiring external servers.
</Note>
